# Gallery photos

Real photos from conferences people found through this list, shown in the **Gallery** section of the root [`README.md`](../../README.md).

## Adding a photo

1. Drop a `.jpg` here with a descriptive name, e.g. `grace-hopper-2026.jpg`.
2. Add an anchor+image inside the `<!-- GALLERY_START -->` / `<!-- GALLERY_END -->` block in the root `README.md`:
   ```html
   <a href="assets/gallery/grace-hopper-2026.jpg"><img src="assets/gallery/grace-hopper-2026.jpg" height="175" alt="Grace Hopper Celebration 2026"></a>
   ```

Prefer landscape photos ~175px tall so the row stays even. Only submit photos you have the right to share.

# Contributor photos

Photos shown on the **Contributors** wall in the root [`README.md`](../../README.md).

## Adding yours

1. Add a square image here named after you, e.g. `firstname.jpg` (roughly 115×115, square crops best).
2. Add a cell to the Contributors table in the root `README.md`:
   ```html
   <td align="center">
     <a href="https://www.linkedin.com/in/your-handle">
       <img src="assets/contributors/firstname.jpg" width="115" height="115" alt="Your Name"><br>
       <sub><b>Your Name</b></sub>
     </a>
   </td>
   ```

Use a square image so it renders cleanly at 115x115 (e.g. `jose.jpg` is 460x460).
